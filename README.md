## 概述
看名字就知道，这个包住要实现了oss的流协议封装，因为是基于官方sdk，也包含了官方sdk的全部功能，应该说，这个包，是在官方的sdk基础上增加流协议功能。

## sdk文档
[Aliyun OSS SDK](https://github.com/medz/aliyun-oss-php-sdk/blob/master/README.md)

## Composer
```shell
composer require medz/oss-stream-wrapper
```
